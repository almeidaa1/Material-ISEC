  ~~Navegaçao interna(No prório site)~~   ex:. Ir de cima da página até baixo da página bastando clicar num hyperlink na
              (Teleport)		  	parte de cima da página WEB para ir para baixo	

1º Damos um nome à secção onde queremos que ao clicar no link se teleporte para lá
 	ex:. <a name="bottom"></a>

2º Na secçao onde se encontra o tal link, para que ao clicar-mos nele a página se teleporte para baixa 
	ex:. <a href="#bottom">CLIQUE AQUI E VÁ PARA BAIXO</a>

3º Abre o site clique no link e vai para baixo
